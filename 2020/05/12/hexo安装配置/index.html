博客搭建：本教程在deepin环境中搭建，Windows或其他系统可参照本教程

安装git

1. 安装git

sudo apt-get install git

2. 配置用户名和邮箱

git config --global user.name "用户名"

git config --global user.email "建议与github用户名相同"

3. 生成ssh公匙、私匙

ssh-keygen -t rsa

三次回车

4. 查看公匙

sudo gedit /home/.ssh/id_rsa.pub

5. 复制公匙到github

登录github-> settings图标

SSH key-> Add SSH key->

填写SSH key的名称,将拷贝的~/.ssh/id_rsa.pub文件内容粘帖-> add key”按钮添加。

6. 测试git

ssh -T git@github.com`
出现类似下面提示，说明git与github关联成功
Hi LaTourEiffel12138! You've successfully authenticated, but GitHub does not provide shell access.

安装node.js

1. 下载

  • 下载地址：http://nodejs.cn/download/current/
  • 选择适合版本

  • 下载对应的.gz文件（node-v10.15.3.tar.gz）

2. 解压到制定目录

sudo tar -zxvf node-v10.15.3-linux-x64.tar.gz -C /usr/local/

3. 创建软链，在任意目录都可以使用

sudo ln -s /usr/local/nodeXXX/bin/node /usr/local/bin/
sudo ln -s /usr/local/nodeXXX/bin/npm  /usr/local/bin/

4. 检验是否成功

node -v
npm -v

安装hexo

1. 安装cnpm

用 npm 安装话经常出现卡住而导致无法正常安装，解决办法就是修改 npm 的安装源，这里选择淘宝 NPM 镜像

sudo npm config set registry https://registry.npm.taobao.org
sudo ln -s /你的nodejs安装目录中的bin/cnpm /usr/local/bin/cnpm

2. 安装

cnpm install -g hexo-cli

sudo ln -s /你的nodejs安装目录中的bin/hexo /usr/local/bin/hexo

3. 查看版本信息

hexo -v

4. 新建Blog文件夹

选定一个位置，新建Blog文件夹，用于存放hexo

cd /media/wgp/e6xxxxxxxxxxx/
sudo mkdir blog
cd blog

5.初始化hexo

hexo init
cnpm initall

6. 查看目录文件

node_modules: 依赖包
scaffolds：生成文章的一些模板
source：用来存放你的文章
themes：主题
** _config.yml: 博客的配置文件**

7. 编译，执行

sudo hexo clean && sudo hexo g && sudo hexo s

浏览器地址输入localhost:4000预览查看

PS：

hexo 安装配置完
需要配置主题可以等我下一篇教程
